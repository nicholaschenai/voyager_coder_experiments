
[description for function: sum_of_unique_numbers]
This function calculates the sum of all unique numbers in a given list. It first creates a frequency dictionary to count the occurrences of each number. Then, it iterates through the dictionary to sum only those numbers that appear exactly once. Finally, it returns the computed sum of these unique numbers. The function effectively filters out duplicates before performing the summation. Overall, it provides a way to identify and sum distinct elements from the input list.

[end of description]
